# 💕 Will You Be My Girlfriend?

A cute and playful interactive webpage to ask someone special to be your girlfriend!

**Live Demo:** https://simp-ux.github.io/KMDL/

## Features

- ✨ Beautiful gradient background with smooth animations
- 💕 Floating hearts and glowing effects
- 🎯 Interactive "Yes" button that triggers confetti celebration
- 😅 Playful "No" button that tries to escape when clicked
- 📱 Fully responsive design that works on all devices
- 🎨 Cute animations and transitions throughout

## How to Use

### Local Preview
Simply open `index.html` in your web browser to preview the webpage.

### Deploy to GitHub Pages

1. **Create a GitHub Repository**
   - Go to [github.com/new](https://github.com/new)
   - Name it (e.g., `will-you-be-my-gf` or keep it as `KMDL`)
   - Make it public
   - Click "Create repository"

2. **Push This Folder to GitHub**
   ```bash
   cd /home/ros2/KMDL
   git init
   git add .
   git commit -m "Initial commit: cute ask-out webpage"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click "Settings"
   - Scroll to "GitHub Pages"
   - Select "Deploy from a branch"
   - Choose `main` branch
   - Click "Save"

4. **Access Your Website**
   - Your site will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME`
   - Or if you named the repo `YOUR_USERNAME.github.io`, it will be at `https://YOUR_USERNAME.github.io`
